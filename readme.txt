Automatically formats inputted KeySAVe data as a Reddit markdown table or as csv.

This website is a semi-clone of [SVExDB](https://cu3po42.github.io/SVExDB-Lite/) and relies on [data](https://raw.githubusercontent.com/Cu3PO42/SVEx-Crawler/gh-pages/tsvs.json) scraped via [SVEx-Crawler](https://github.com/Cu3PO42/SVEx-Crawler).
